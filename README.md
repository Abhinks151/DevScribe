# Blog App (React + Firebase)

A simple blog application built using **React** for the frontend and **Firebase** as the Backend-as-a-Service (BaaS).

## About the Project

This project was created to gain hands-on experience with **Firebase**, mainly focusing on **Authentication** and **Cloud Firestore**, while building a real-world React application.

It demonstrates user authentication, secure data storage, and basic CRUD operations, helping to understand how React integrates with Firebase for scalable and serverless app development.

## Tech Stack

- React
- Firebase Authentication
- Cloud Firestore
